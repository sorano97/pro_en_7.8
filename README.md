# 神山町 危険箇所マップ

神山町周辺の危険箇所を地図上で確認できるWebアプリです

交通量が多い場所、暗い道、不審者情報、見通しが悪い場所、事故、自然災害などの情報をGeoJSON形式で管理し、Leafletを使って地図上に可視化しています

---

## 1. アプリの概要

「神山町 危険箇所マップ」は、地域に存在する危険な場所を地図上で分かりやすく確認するためのWebアプリです

学生、保護者、地域住民などが、通学路や夜道、交通量の多い場所などを事前に確認し、自分たちの身の回りの安全について考えるきっかけを作ることを目的としています

危険箇所ごとに、次のような情報を確認できます

- 場所名
- 緯度・経度
- 危険の種類
- 危険度
- 説明文
- 報告日時
- 投稿者区分
- 近くの避難場所
- 近くの交番・警察署
- 相談・通報先

危険情報を一覧として表示するだけではなく、地図上にマーカーとして表示することで、「どこに」「どのような危険があるのか」を直感的に確認できるようにしました

---

## 2. 主な機能

### 2.1 危険箇所の地図表示

GeoJSONファイルに保存された危険箇所データを読み込み、Leafletを使って地図上にマーカーとして表示します

危険箇所をクリックすると、場所名、危険の種類、危険度、説明文などの詳細情報を確認できます

### 2.2 危険の種類ごとの色分け

危険の種類によってマーカーの色を変更し、視覚的に区別できるようにしています

対応している危険の種類は次のとおりです

- 交通量が多い
- 暗い道
- 不審者
- 見通しが悪い
- 事故
- 自然災害
- その他

### 2.3 危険度の表示

各危険箇所には1〜5の5段階で危険度が設定されています

地図上のマーカーには危険度の数字を表示し、詳細画面では星による表示も行います

### 2.4 フィルター機能

危険の種類ごとに、地図上に表示する情報を絞り込めます

複数の種類を同時に選択することもできます

例えば、「暗い道」と「不審者」だけを同時に表示するといった使い方ができます

### 2.5 検索機能

場所名、説明文、相談・通報先などの文字情報から危険箇所を検索できます

検索結果に一致した危険箇所のみを地図上に表示します

### 2.6 現在地表示

ブラウザの位置情報機能を利用して、自分の現在地を取得できます

取得した現在地は地図上に表示され、緯度と経度も自動で入力されます

### 2.7 地図上から緯度・経度を取得

地図上の好きな地点をクリックして、危険箇所として登録したい場所を選択できます

選択すると、その場所の緯度と経度が入力欄に自動で反映されます

また、地図上に表示されたピンをドラッグすることで、位置を細かく調整できます

### 2.8 マーカークラスタリング

危険箇所が多く集まっている場所では、複数のマーカーを1つにまとめて表示します

まとめられたマーカーには件数が表示され、地図を拡大すると個別の危険箇所を確認できます

### 2.9 ひらがな表示

漢字を読むことが難しい人にも利用しやすいように、画面上の文字をひらがな中心の表示に切り替える機能があります

### 2.10 英語表示

日本語と英語を切り替えて利用できます

日本語を母語としない利用者にも使いやすいWebアプリを目指しました
